# Notebooks

This directory contains Jupyter notebooks used for:

- Exploratory data analysis (EDA)
- Data profiling
- Experimentation
- Visualizations
- Prototyping transformations

Notebooks are for exploration and should not replace production scripts.

Project 1: Nashville Housing — End‑to‑End Data Pipeline (SQL + DuckDB)  
This project delivers a complete data workflow for the Nashville Housing dataset. It covers:

Importing raw Excel data

Cleaning and standardizing inconsistent fields

Splitting and restructuring address components

Handling missing values

Normalizing categorical fields

Removing duplicates using window functions

Creating a final cleaned dataset for analysis

The project is structured using a professional data‑engineering layout (raw → processed) and is fully reproducible inside Jupyter Notebook using DuckDB SQL.
